## 기술 스택
* 프레임워크: Android Jetpack, RxJava2
* 라이브 방송: Agora SDK
* 네트워크: Retrofit, OkHttp, Gson

## 실행 환경 및 방법
* Android Studio 4.0 이상
* Android Nougat(SDK 24) Device 이상
* strings.xml 파일에 agora_app_id 리소스에 Agora App ID 입력
